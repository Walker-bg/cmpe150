Question E1:

Recursive String Iteration

Please read important information first.

Description

You are given a string which includes at least one digit.
Write a recursive function that finds the sum of the digits in the string.
Ignore any other character that is not a digit.

You can ONLY use the following:

  • functions: len() ord()
  • operators: + - * < > <= >=
  • for indexing: [< dig >:< dig >] where < dig > corresponds to a digit
  • keywords: if, else, return
  • for and while loops are not allowed

Important:

You can use the space given to you to write code as you wish (such as writing your own function or functions). Do not take another input.

Input part is handled for you, you can directly use variable s.

length of the string <= 30

Warning: You are not allowed to use imports, or anything that has not been covered in class. If you do not abide by these rules you may get no points.

Examples:
INPUT OUTPUT
11b4A2et5 13
1sajksa..[-=-291b4A2et5 24
